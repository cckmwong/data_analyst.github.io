The Best Place to Live in England
(Power BI)

The aim of this report is for new immigrants to find their best place to live in England, in respect of i) safety, ii) housing prices, iii) population and income deprivation and iv) school performance. At the end of the report, a final score will be calculated for each selected local authority, and users can find the ideal place to live in their selections.
Besides, there is correlation analysis to evaluate the factors which affect the results of key stage 2 (KS2) and key stage 4 (KS4) of the schools in England.

Data Extraction with SQL

Here is data extraction from .csv files downloaded from the GOV.UK website, which are three separate tables concerning individual school information, KS2 results and KS4 results. Data cleaning processes include i) dropping unrelated rows, ii) merging tables, iii) handling null values, iv) checking duplicates, etc. After executing the SQL codes in Microsoft SQL Server, a new and cleaned table is generated and is ready for use in subsequent data analysis.
